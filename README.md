# Capstone Assembler Project

**Date:** Dec 19, 2025

## Description: This project is an academic school project to teach us about different programming paradigms.

## Build:

CMAKE or

```bash
gcc src/fileio.c src/lexer.c src/log.c src/lookup_gen.c src/main.c src/parser.c -Iinclude -Isvec -o masm
```
